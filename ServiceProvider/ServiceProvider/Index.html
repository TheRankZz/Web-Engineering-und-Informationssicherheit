Webservice Gruppe 1

API Beschreibung
POST /{user} Einen neuen Benutzer registrieren
GET /{user} Einen Benutzer holen
GET /{user}/publickey Einen öffentlichen Schlüssel vom Benutzer holen
DELETE /{user} Einen Benutzer löschen
POST /{user}/message Eine Nachricht verschicken
GET /{user}/message Eine Nachricht vom Benutzer holen
